# 🏗️ Week 1 — Design Patterns, DSA, PL-SQL & TDD

Welcome to **Week 1** of the Java FSE curriculum! This week covers fundamental software engineering design patterns, data structures and algorithms, database programming with PL/SQL, and unit testing with logging.

## 📁 Modules Included

1. **[Module 1 - Design Patterns and Principles](./Module%201%20-%20Design%20Patterns%20and%20Principles/)**: The 11 core Gang of Four (GoF) design patterns (Singleton, Factory, Builder, Observer, etc.) and SOLID principles.
2. **[Module 2 - Data Structures and Algorithms](./Module%202%20-%20Data%20Structures%20and%20Algorithms/)**: Implementation of sorting algorithms (bubble, quick, merge), searching (linear, binary), arrays, and linked lists.
3. **[Module 3 - PL SQL Programming](./Module%203%20-%20PL%20SQL%20Programming/)**: PL/SQL blocks, variables, control structures, cursors, stored procedures, functions, packages, and triggers.
4. **[Module 4 - Test driven development and Logging framework](./Module%204%20-%20Test%20driven%20development%20and%20Logging%20framework/)**: Unit testing using JUnit 5, mocking dependencies with Mockito, and application logging using SLF4J and Lombok annotations.

## 🎯 Learning Objectives

* Apply SOLID principles and GoF design patterns for clean code architecture.
* Analyze time and space complexity (Big-O) and optimize operations with proper data structures.
* Automate database logic using PL/SQL procedures, triggers, and functions.
* Write unit tests using JUnit 5 and Mockito while setting up logging appenders.
