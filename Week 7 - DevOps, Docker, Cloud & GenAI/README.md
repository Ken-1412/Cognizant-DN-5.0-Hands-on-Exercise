# ☁️ Week 7 — DevOps, Docker, Cloud & GenAI Fundamentals

Welcome to **Week 7**! This week covers modern deployment DevOps practices, containerization with Docker, cloud platform deployment, and Generative AI integration fundamentals.

## 🎯 Topics & Learning Objectives

* **Docker & Containerization**: Writing Dockerfiles, creating container images, running multi-container applications using Docker Compose.
* **DevOps & Cloud**: Deploying full-stack Java applications to cloud platforms (AWS / Azure) with infrastructure best practices.
* **Generative AI Fundamentals**: Prompt engineering, integrating LLM API endpoints into Java microservices, and AI-assisted development workflows.
