# 🎨 Week 1 — Frontend Basics

Welcome to **Week 1** of the Cognizant Digital Nurture 5.0 program! This week focuses on the fundamentals of building web interfaces.

## 📁 Modules Included

1.  **[Module 1-HTML 5](./Module%201-HTML%205/)**: Standard HTML structures, semantic elements, and forms.
2.  **[Module 1-CSS3](./Module%201-CSS3/)**: Styling interfaces using colors, grids, flexbox, and selectors.
3.  **[Module 1-Bootstrap5](./Module%201-Bootstrap5/)**: Responsive web design using Bootstrap's grid system and pre-built components.

## 🎯 Learning Objectives

*   Build well-structured, semantic web pages.
*   Implement layout styling and responsive design using modern CSS.
*   Leverage Bootstrap to build modern, mobile-friendly user interfaces quickly.

## 🧭 How to Test & Review

Open any of the `.html` files in a browser to see your styled layouts and responsive designs in action!
