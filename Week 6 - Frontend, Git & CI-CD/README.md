# 🚀 Week 6 — Frontend (Angular & React), Git & CI/CD

Welcome to **Week 6**! This week focuses on advanced frontend component integration, version control using Git, and continuous integration/continuous deployment (CI/CD) pipelines.

## 🎯 Topics & Learning Objectives

* **Angular / React Advanced Integration**: Component state management, routing, form validation, HTTP client services connecting to Spring REST backends.
* **Git Version Control**: Branching strategies, pull requests, merge conflict resolution, and release tagging.
* **CI/CD Pipelines**: Automated build, test, and release pipelines (GitHub Actions / Jenkins / Azure DevOps).
