# 🚀 Cognizant Digital Nurture 5.0 — Java Full Stack Engineer

Welcome to my learning repository for the Cognizant Digital Nurture 5.0 (DN 5.0) Java FSE program. This repository has been structured week-by-week so that the coursework can be reviewed, completed, and uploaded in a balanced sequence.

---

## 📅 Week-Wise Learning Path

Here is the 7-week curriculum map matching the completed exercises in this repository:

| Week | Title | Modules Covered | Key Topics | Status |
|---|---|---|---|---|
| **[Week 1](./Week%201%20-%20Frontend%20Basics/)** | Frontend Basics | HTML5, CSS3, Bootstrap 5 | Semantic tags, flexbox/grids, responsive layouts | ✅ Complete |
| **[Week 2](./Week%202%20-%20Client%20Logic%20&%20Core%20Java/)** | Client Logic & Core Java | JavaScript, Core Java OOP | DOM manipulation, Collections, Multithreading, Exception Handling | ✅ Complete |
| **[Week 3](./Week%203%20-%20Databases & Design Patterns/)** | Databases & Design Patterns | MySQL SQL, SOLID & GoF Design Patterns | SQL Joins, Aggregations, Creational/Structural/Behavioral patterns | ✅ Complete |
| **[Week 4](./Week%204%20-%20DSA & Advanced Database/)** | DSA & Advanced Database | Arrays, Lists, Sort/Search, PL/SQL | Binary search, Quick/Merge sort, Stored procedures, triggers | ✅ Complete |
| **[Week 5](./Week%205%20-%20TDD, Logging & Core Spring/)** | TDD, Logging & Core Spring | JUnit 5, Mockito, SLF4J, Spring IoC/DI | Unit tests, mock injection, logging appenders, dependency injection | ✅ Complete |
| **[Week 6](./Week%206%20-%20Data Integration & REST APIs/)** | Data Integration & REST APIs | Spring Data JPA, Hibernate, REST Controllers | Entity mapping, JPA CRUD, Pageable finders, HTTP verbs, JSON mappings | ✅ Complete |
| **[Week 7](./Week%207%20-%20Security & Project Integration/)** | Security & Project Integration | Spring Security, JWT, Library Management | Auth filters, token claims, full-stack MVC project integration | ✅ Complete |
