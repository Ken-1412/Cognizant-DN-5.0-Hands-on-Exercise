# 🚀 Cognizant Digital Nurture 5.0 — Java Full Stack Engineer

Welcome to the learning repository for the **Cognizant Digital Nurture 5.0 (DN 5.0) Java FSE** program. This repository is structured week-by-week according to the official curriculum track schedule.

---

## 📅 Week-Wise Learning Path (Java FSE Skill)

Here is the 7-week curriculum map matching the coursework in this repository:

| Week | Title | Key Topics Covered | Status |
|---|---|---|---|
| **[Week 1](./Week%201%20-%20Design%20Patterns,%20DSA,%20PL-SQL%20&%20TDD/)** | Design Patterns, DSA, PL-SQL & TDD | Design Patterns & Principles, Data Structures & Algorithms, PL/SQL Programming, Test-Driven Development (JUnit/Mockito) & Logging | ✅ Complete |
| **[Week 2](./Week%202%20-%20Spring%20Core,%20JPA%20&%20Hibernate/)** | Spring Core, JPA & Hibernate | Spring Core IoC & DI, Maven, Spring Data JPA with Spring Boot, Hibernate, Core Java & SQL foundations | ✅ Complete |
| **[Week 3](./Week%203%20-%20Spring%20REST/)** | Spring REST | Spring REST APIs using Spring Boot 3, REST Controllers, HTTP verbs & Exception Mapping | ✅ Complete |
| **[Week 4](./Week%204%20-%20Code%20Quality%20&%20Microservices/)** | Code Quality, Microservices & Security | Code Quality & SonarQube, Microservices (Spring Boot 3 + Spring Cloud), Spring Security & JWT Authentication | ✅ Complete |
| **[Week 5](./Week%205%20-%20Frontend%20\(Angular%20&%20React\)/)** | Frontend Basics, Angular & React | HTML5, CSS3, Bootstrap 5, JavaScript (ES6+), Angular / React UI foundations | ✅ Complete |
| **[Week 6](./Week%206%20-%20Frontend,%20Git%20&%20CI-CD/)** | Frontend, Git & CI/CD | Angular / React SPA components, Git Version Control, Branching & CI/CD Pipelines | ✅ Complete |
| **[Week 7](./Week%207%20-%20DevOps,%20Docker,%20Cloud%20&%20GenAI/)** | DevOps, Docker, Cloud & GenAI | DevOps practices, Docker containerization, Cloud deployment & GenAI fundamentals | ✅ Complete |
