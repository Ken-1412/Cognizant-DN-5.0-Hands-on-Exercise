# 🗄️ Week 3 — Databases & Design Patterns

Welcome to **Week 3**! This week focuses on relational databases using MySQL and the foundational software design patterns.

## 📁 Modules Included

1.  **[Module 2-ANSI SQL Using MySQL](./Module%202-ANSI%20SQL%20Using%20MySQL/)**: Designing schemas, writing queries, joins, subqueries, and aggregations.
2.  **[Module 1 - Design Patterns and Principles](./Module%201%20-%20Design%20Patterns%20and%20Principles/)**: The 11 core Gang of Four (GoF) design patterns (Singleton, Factory, Builder, Observer, etc.) and SOLID principles.

## 🎯 Learning Objectives

*   Model relational database schemas and write complex SQL queries.
*   Understand SOLID principles (SRP, OCP, LSP, ISP, DIP).
*   Use design patterns to write clean, reusable, and maintainable Java code.
