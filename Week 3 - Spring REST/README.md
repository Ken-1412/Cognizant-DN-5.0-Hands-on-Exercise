# 🌐 Week 3 — Spring REST

Welcome to **Week 3**! This week focuses on building scalable RESTful web services using Spring Boot 3.

## 📁 Modules Included

1. **[spring-rest-handson](./spring-rest-handson/)**: Hands-on project building REST controllers, handling HTTP verbs (GET, POST, PUT, DELETE), response entity formatting, request parameters, path variables, and exception mapping.
2. **[Module7_REST_README.md](./Module7_REST_README.md)**: Detailed reference guide for REST architecture, Spring REST annotations, and API best practices.

## 🎯 Learning Objectives

* Design and implement client-agnostic RESTful API endpoints using Spring Boot 3.
* Handle JSON data binding, URI paths, query parameters, and custom HTTP response statuses.
* Implement centralized exception handling using `@ControllerAdvice` and `@ExceptionHandler`.
