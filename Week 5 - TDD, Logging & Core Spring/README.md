# 🧪 Week 5 — TDD, Logging & Core Spring

Welcome to **Week 5**! This week introduces modern software development practices (testing and logging) alongside the core Spring framework.

## 📁 Modules Included

1.  **[Module 4 - Test driven development and Logging framework](./Module%204%20-%20Test%20driven%20development%20and%20Logging%20framework/)**: Unit testing using JUnit 5, mocking dependencies with Mockito, and logging using SLF4J and Lombok annotations.
2.  **[Module 5 - Spring Core Maven](./Module%205%20-%20Spring%20Core%20Maven/)**: Spring IoC Container, Dependency Injection (Constructor vs. Setter), Autowiring, Bean scopes, and project builds using Maven.

## 🎯 Learning Objectives

*   Write robust unit tests using Mockito and JUnit to support Test-Driven Development (TDD).
*   Implement application logging levels (INFO, DEBUG, ERROR) using SLF4J.
*   Understand Spring's IoC container and configure beans using XML and annotations.
