# 📊 Week 4 — DSA & Advanced Database

Welcome to **Week 4**! This week shifts focus to data structures/algorithms and database programming using PL/SQL.

## 📁 Modules Included

1.  **[Module 2 - Data Structures and Algorithms](./Module%202%20-%20Data%20Structures%20and%20Algorithms/)**: Implementing sorting algorithms (bubble, quick, merge), searching (linear, binary), arrays, and linked lists.
2.  **[Module 3 - PL SQL Programming](./Module%203%20-%20PL%20SQL%20Programming/)**: PL/SQL blocks, variables, control structures, cursors, stored procedures, functions, packages, and triggers.

## 🎯 Learning Objectives

*   Analyze algorithmic time and space complexity (Big-O notation).
*   Optimize data operations using the right data structure (arrays, lists).
*   Build automated database logic using stored procedures, triggers, and functions.
