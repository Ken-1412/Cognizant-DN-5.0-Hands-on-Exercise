# 🎨 Week 5 — Frontend (Angular & React)

Welcome to **Week 5**! This week focuses on client-side frontend development fundamentals and SPA frameworks (Angular / React).

## 📁 Modules Included

1. **[Module 1-HTML 5](./Module%201-HTML%205/)**: Semantic HTML structure, forms, tables, and media elements.
2. **[Module 1-CSS3](./Module%201-CSS3/)**: Styling, layout positioning, Flexbox, CSS Grid, and responsive design.
3. **[Module 1-Bootstrap5](./Module%201-Bootstrap5/)**: Responsive UI components, navigation, grid container system, and utility classes.
4. **[Module 1-JavaScript](./Module%201-JavaScript/)**: ES6 syntax, DOM manipulation, event handling, promises, and async/await for dynamic logic.
5. **[Module 1-JavaScript Build Tools](./Module%201-JavaScript%20Build%20Tools/)**: Configuring Webpack bundler and Babel transpiler for modern JavaScript development.

## 🎯 Learning Objectives

* Master core web fundamentals (HTML5, CSS3, Bootstrap 5) for responsive UI layout design.
* Implement dynamic client logic using modern JavaScript (ES6+).
* Build modern Single Page Applications (SPA) using Angular or React frameworks.
