# 🛡️ Week 4 — Code Quality, Microservices & Security

Welcome to **Week 4**! This week focuses on Code Quality (SonarQube), Microservices architecture using Spring Boot 3 & Spring Cloud, and API Security integration using Spring Security & JSON Web Tokens (JWT).

## 📁 Modules Included

1. **[JWT-handson](./JWT-handson/)**: Restricting access to REST APIs using Spring Security filters, authentication managers, login endpoints, token generation, and role-based authorization.
2. **[library-management](./library-management/)**: End-to-end full-stack integration project integrating Spring Boot, Spring Security, JWT, Spring Data JPA, and REST Controllers.
3. **[Module7_JWT_README.md](./Module7_JWT_README.md)**: Implementation guide for JWT token claims, security filter chains, and authorization patterns.

## 🎯 Learning Objectives

* Configure Spring Security filter chain to secure microservices endpoints.
* Implement JWT token generation, parsing, and stateless request verification.
* Apply SonarQube principles for static code analysis, code coverage, and quality gates.
* Understand Microservices architecture, API gateways, and service discovery with Spring Cloud.
