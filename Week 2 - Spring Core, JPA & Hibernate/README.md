# 🍃 Week 2 — Spring Core, JPA & Hibernate

Welcome to **Week 2**! This week focuses on Core Spring dependency injection, Maven build management, and persistent relational data storage using Spring Data JPA & Hibernate.

## 📁 Modules Included

1. **[Module 3-Core Java](./Module%203-Core%20Java/)**: Core Java foundations including OOP principles, Collections, Multithreading, Exception handling, and File I/O.
2. **[Module 2-ANSI SQL Using MySQL](./Module%202-ANSI%20SQL%20Using%20MySQL/)**: Relational schema design, SQL queries, joins, subqueries, and aggregations using MySQL.
3. **[Module 5 - Spring Core Maven](./Module%205%20-%20Spring%20Core%20Maven/)**: Spring IoC Container, Dependency Injection (Constructor vs. Setter), Autowiring, Bean scopes, and project builds using Maven.
4. **[Module 6 - Spring Data JPA with Spring Boot, Hibernate](./Module%206%20-%20Spring%20Data%20JPA%20with%20Spring%20Boot,%20Hibernate/)**: Entity mapping, repositories, JpaRepository CRUD operations, custom query methods, auditing, sorting, and pagination.

## 🎯 Learning Objectives

* Configure Spring IoC container and manage bean lifecycles using XML and annotations.
* Map Java domain entities to database tables using JPA/Hibernate without raw SQL.
* Perform database CRUD, pagination, and custom queries using Spring Data JPA.
