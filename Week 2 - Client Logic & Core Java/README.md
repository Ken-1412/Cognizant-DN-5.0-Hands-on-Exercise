# ⚡ Week 2 — Client Logic & Core Java

Welcome to **Week 2**! This week bridges dynamic frontend logic with backend Java programming.

## 📁 Modules Included

1.  **[Module 1-JavaScript](./Module%201-JavaScript/)**: Dynamic programming, DOM manipulation, ES6 features, and event handling.
2.  **[Module 3-Core Java](./Module%203-Core%20Java/)**: Core Java concepts including Object-Oriented Programming (OOP), Collections framework, Multithreading, Exception Handling, and File I/O.

## 🎯 Learning Objectives

*   Add dynamic features and validation to front-end forms using JavaScript.
*   Master core Java structures (lists, maps, inheritance) to prepare for backend frameworks.
*   Understand robust programming practices such as handling exceptions.
